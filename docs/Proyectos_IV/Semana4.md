# Semana 4

## 1) Diagrama de Gantt

Durante esta semana construimos un diagrama de Gantt para ordenar de forma clara las actividades necesarias para completar el proyecto y asignarles un intervalo de tiempo específico a cada una, con la intención de hacer más eficiente el desarrollo del producto final.

En esta sección se muestra el diagrama de Gantt del proyecto. Esta herramienta permite organizar las actividades en una línea de tiempo, indicando cuándo inicia y termina cada tarea. Gracias a esto podemos visualizar el orden de las actividades, qué cosas se hacen en paralelo y el tiempo estimado para llegar al producto final, facilitando el seguimiento y control del avance del proyecto.
